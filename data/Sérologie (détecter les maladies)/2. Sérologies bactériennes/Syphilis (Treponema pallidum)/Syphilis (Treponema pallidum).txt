Analyses Sérologiques : Sérologies Bactériennes

1. Syphilis (Treponema pallidum)
1.1. Définition
La syphilis est une maladie infectieuse chronique, sexuellement transmissible, causée par la bactérie Treponema pallidum, un spirochète. Elle se caractérise par plusieurs stades d’évolution cliniques, notamment primaire, secondaire, latent et tertiaire. Sans traitement, la syphilis peut entraîner des complications graves, affectant divers organes, notamment le cœur, le système nerveux et les os.

La syphilis est souvent surnommée la "grande imitatrice" en raison de ses symptômes qui peuvent mimer d'autres maladies. Elle se transmet principalement par contact sexuel, mais aussi de la mère à l'enfant lors de la grossesse (syphilis congénitale).

1.2. Tests Sérologiques pour la Syphilis
Les tests sérologiques sont utilisés pour diagnostiquer l'infection par Treponema pallidum. Ils sont également utilisés pour dépister et surveiller l'évolution de la syphilis. Les tests sérologiques se divisent en deux grandes catégories :

Tests non tréponémiques (tels que le VDRL et le RPR) : Ces tests détectent les anticorps produits par l'organisme en réponse à la lésion causée par l'infection.
Tests tréponémiques (tels que le FTA-ABS et le TPHA) : Ces tests détectent directement les anticorps dirigés contre la bactérie Treponema pallidum.
1.2.1. Tests Non-Tréponémiques (VDRL / RPR)
Test de dépistage utilisé pour identifier les anticorps non spécifiques produits par le corps en réponse à l'infection (anticorps réactifs au cardiolipine).
Utilisation : Utilisé comme test de dépistage et pour surveiller l'efficacité du traitement. Un résultat positif nécessite des tests tréponémiques pour confirmer le diagnostic.
Valeur normale : Négatif
Signification clinique :

Un résultat positif peut indiquer une infection récente ou active, mais peut aussi être retrouvé dans d'autres maladies (lupus, grossesse, infections virales ou bactériennes, etc.). Il peut aussi être positif en cas de syphilis traitée, dans ce cas il diminue avec le traitement.
Un résultat négatif n’élimine pas la possibilité de syphilis, en particulier au stade primaire ou très tôt dans l'infection.
1.2.2. Tests Tréponémiques (FTA-ABS / TPHA)
Test spécifique : Ces tests détectent les anticorps dirigés contre Treponema pallidum.
Le FTA-ABS (Fluorescent Treponemal Antibody Absorption Test) et le TPHA (Treponema pallidum Hemagglutination Assay) sont les tests les plus couramment utilisés.
Utilisation : Ces tests sont utilisés pour confirmer le diagnostic de syphilis après un résultat positif à un test non tréponémique.
Valeur normale : Négatif
Signification clinique :

Positif : Indique une infection présente ou passée par Treponema pallidum. Les résultats positifs persistent pendant toute la vie, même après un traitement efficace.
Négatif : N’indique pas de syphilis. Cependant, un test tréponémique négatif ne permet pas d'exclure une infection très récente, où les anticorps spécifiques peuvent encore être indétectables.
1.3. Valeurs Normales et Interprétation Clinique
Les tests sérologiques permettent de définir le stade de la syphilis et d'interpréter les résultats comme suit :

Syphilis primaire (stade 1) :

Symptômes : Ulcère (chancre) indolore au site d'inoculation.
Tests sérologiques :
VDRL/RPR : Peut être négatif ou faiblement positif.
FTA-ABS/TPHA : Positif, mais encore trop tôt dans l'infection pour que les tests non tréponémiques soient largement positifs.
Syphilis secondaire (stade 2) :

Symptômes : Éruption cutanée, lésions muqueuses, symptômes systémiques (fièvre, malaise).
Tests sérologiques :
VDRL/RPR : Très positif.
FTA-ABS/TPHA : Positif.
Syphilis latente (stade 3) :

Symptômes : Absence de symptômes cliniques.
Tests sérologiques :
VDRL/RPR : Peut être positif à titre faible, mais se stabilise.
FTA-ABS/TPHA : Positif.
Syphilis tertiaire (stade 4) :

Symptômes : Complications graves comme des lésions nerveuses (neurosyphilis), cardiovasculaires (syphilis cardiovasculaire), et osseuses.
Tests sérologiques :
VDRL/RPR : Peut être positif ou négatif (en fonction de l'état de l'infection).
FTA-ABS/TPHA : Positif à vie.
1.4. Signification Clinique
Diagnostic précoce : Un diagnostic rapide est essentiel pour prévenir les complications graves de la syphilis, telles que la neurosyphilis, la syphilis cardiovasculaire et la syphilis congénitale (transmission de la mère à l'enfant).
Complications : Si non traitée, la syphilis peut progresser vers un stade tardif où des complications sérieuses peuvent apparaître, affectant le cœur (aortite syphilitique), le système nerveux (neurosyphilis) et les os (syphilis gommée).
Syphilis congénitale : La syphilis transmise de la mère à l’enfant peut entraîner des malformations, des anomalies neurologiques et des décès intra-utérins. Un dépistage systématique de la syphilis chez les femmes enceintes est essentiel.
1.5. Indications Cliniques pour les Tests Sérologiques de la Syphilis
Les tests sérologiques pour la syphilis sont recommandés dans les situations suivantes :

Dépistage des personnes à risque : Notamment les personnes ayant plusieurs partenaires sexuels, les hommes ayant des rapports sexuels avec des hommes, les personnes vivant avec le VIH, les femmes enceintes, etc.
Symptômes cliniques suspects : En présence de symptômes évocateurs (chancre, éruption cutanée, lésions muqueuses).
Surveillance des patients traités : Pour évaluer l'efficacité du traitement (les titres de VDRL/RPR devraient diminuer avec le temps si le traitement est efficace).
Contrôle dans les contacts sexuels : Tous les partenaires sexuels doivent être dépistés et traités pour éviter la réinfection.
1.6. Traitement de la Syphilis
La syphilis est facilement traitable avec des antibiotiques, généralement de la pénicilline. Le traitement est efficace à tout stade de l'infection, mais plus les stades sont avancés, plus les complications peuvent être graves, même après traitement.
